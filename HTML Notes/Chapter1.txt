                                                      INTRODUCTION TO HTML -  (Chapter - 1)

-  INTRO - 
-> HTML stands for HyperText Markup Language.
-> HTML is used to create web pages and web applications.
-> HTML is widely used language on the web.
-> Technically, HTML is a Markup language rather than a programming language.

-> HyperText - HyperText Is a Text That Has A Link Within It. HyperText Is Way To Link Two Or More Web Pages. 
-> Markup language - is a computer language that is used to apply layout and formatting conventions to a text document.


- What Is HTML?
1. HTML describes the structure of a Web page.
2. HTML consists of a series of elements.
3. HTML elements tell the browser how to display the content.
4. HTML is the standard markup language for creating Web pages.
